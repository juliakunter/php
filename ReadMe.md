# WebCommunication  
**Author:** Kunter  Julia
<br>
**LBS Eibiswald | 3aAPC | LG2**

Diese Anwendung zeigt den **Kommunikationsablauf zwischen Webserver und Browser** auf einfache Weise.  
Sie dient als **Lern- und Demo-Projekt**, um zu verstehen, wie **Client und Server** über das **HTTP-Protokoll** miteinander sprechen.  

Das Ziel ist es, ein **grundlegendes Verständnis** dafür zu vermitteln, was passiert, wenn man im Browser eine Website aufruft – von der Anfrage bis zur Antwort.

---

## Idee

Ein Browser (Client) sendet eine **Anfrage (Request)** an den Server,  
und der Server gibt eine **Antwort (Response)** zurück.  
So entsteht die Verbindung, über die Webseiten, Daten und Inhalte übertragen werden.

---

## Installation

Um das Projekt lokal auszuführen:

```bash

# In das Projektverzeichnis wechseln
cd WebCommunication

# PHP-Version prüfen
